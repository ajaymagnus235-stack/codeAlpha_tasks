# Language Translation Tool

## Project Description

This is a Language Translation Tool built using Python, Flask, HTML and Google Translator API.

The application allows users to:

- Enter text
- Select a target language
- Translate text instantly
- View translated output

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Deep Translator
