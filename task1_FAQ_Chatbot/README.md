AI FAQ Chatbot

Project Overview

This project is an AI-powered FAQ Chatbot developed using Flask and Natural Language Processing (NLP).

The chatbot answers frequently asked questions by comparing the user's query with stored FAQs using TF-IDF Vectorization and Cosine Similarity.

Features

- Interactive Chat Interface
- FAQ Question Matching
- NLP-Based Similarity Search
- User and Bot Chat Bubbles
- Enter Key Support
- Responsive UI

Technologies Used

- Python
- Flask
- Scikit-Learn
- HTML
- CSS
- JavaScript

Installation

pip install flask scikit-learn
python app.py

Project Structure

- app.py
- faq.json
- templates/index.html
- static/style.css
